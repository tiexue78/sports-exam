# 项目：初二体育知识练习系统

## 项目说明
在线练习系统，面向初二学生，用于体育与健康知识考试的反复练习。手机端为主，需在线部署。

## 技术栈
Next.js + Supabase + Tailwind CSS + Vercel

## 源文件
- `3_北京市初中学业水平考试体育与健康知识试题(1).pdf` — 题目原文
- `选择题、判断答案(1)(1).docx` — 答案

## 开发指引
- 移动端优先设计
- 题目数据需从源文件解析后导入数据库
- 所有数据持久化到 Supabase，不使用 localStorage
- 用户体系极简：姓名即可，重名提示换名
